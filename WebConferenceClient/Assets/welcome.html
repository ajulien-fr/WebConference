This is a welcome page of WebConferenceClient.

WebConferenceClient is a minimal demo application about WCF and UWP.

This app use MVVM Light Toolkit for the Model–View–ViewModel (MVVM) implementation and UWPCommunityToolkit for its controls.

First, run the WCF service hoster for starting the service. After, here, go to the Channel page and entry the address of the WCF service.

Now, you can speak with everything connected to the service.
